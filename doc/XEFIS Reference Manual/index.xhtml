Introduction

XEFIS is modular avionics software designed for remote controlled airplanes. It can work in a flight computer (performing stabilization, flight-envelope protection, autoflight, etc) and in a ground control station, mainly as an interface for you input controls and showing flight instruments.

Hardware

You'll need at least two computers: one for the aircraft and one for the ground control station.

For the aircraft, something ARM-based should be OK. It should have a lot of useful interfaces (several serial ports, IÂ²C, analog inputs, etc.) for external sensors. It doesn't need to be very powerful, something like Raspberry Pi or Beagle Bone Black should work perfectly.

On a ground control station you'll need something more powerful, maybe dual-core Atom mini ITX motherboard. Or you can just use a notebook PC.
