AFCS - Automated Flight Control System systems/afcs

Glossary

FMC is Flight Management Computer.

MCP is Main Control Panel.

Speed modes

Mode on FMA Controls Meaning
TO/GA thrust Use maximum temporary thrust available.
N1 thrust Maintain maximum continuous thrust, configured in FMC.
FMC SPD speed Maintain normal cruise speed, set in FMC.
MCP SPD speed Maintain speed selected on MCP panel.
SPD HOLD speed Maintain speed present at the moment the HOLD mode was engaged.
FMC DESC speed Maintain speed for descend, configured in FMC.
IDLE thrust Maintain minimum thrust.

Buttons

Button FMA message Description
CLB/CON N1 (speed),
CLB/CON (vertical)

Set thrust to N1, maintain vertical speed with pitch. Default vertical speed is configured in FMC.

Disengages when vertical speed on MCP is set to or below 0 fpm.
V/S HOLD V/S

Maintain current vertical speed with pitch.

MCP counter is set to current vertical speed.

If FLCH mode was engaged when the button was pressed, this mode will disengage when selected altitude is reached.

When vertical speed is set to 0 on the counter, the mode disengages. If FLCH was engaged when counter was set to 0, FLCH mode also disengages and altitude mode is set to ALT HOLD.
VS/FPA VS or FPA

Climb or descend with VS or FPA selected on the counter.

Disabling this mode by pressing the button will:

  • enable ALT HOLD mode at current altitude, if FLCH mode was not engaged previously
  • continue in LVL CHG mode with default vertical speed

If FLCH mode was engaged when the button was pressed:

  • if VS or FPA is set to 0 or in the opposite direction than the commanded altitude, FLCH mode will disengage.
  • when altitude selected on ALTITUDE counter is reached, this mode will be deactivated.
