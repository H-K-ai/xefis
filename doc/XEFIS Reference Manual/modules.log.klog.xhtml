KLog - Kernel Ring Buffer health check log/klog

This module searches in Linux kernel ring buffer for problem indicators, then sets appropriate boolean flags, which can be forwarded to the ground station for pilot information. The search is done in case-insensitive manner.

Output

Set to true once oom-killer string is found in kernel ring buffer.

Set to true once i/o error string is found in kernel ring buffer.

Set to true once oops string is found in kernel ring buffer.

Set to true once bug string is found in kernel ring buffer.
